March 27th - April 2nd

This week I tried solving a certain issue regarding the number of participants in a tournament.
This number was always 10 users per tournament, although it seemed that the master could save more people in.
In fact, when I tried add new users with a higher score that the last ones, they appeared to smash them, making it so that they wouldn't appear again ever.
This was a sign that it wasn't a problem on the database but rather the Search methods.

After trying lots of patchs, I was finnally able to solve it by just telling python that the size of the data that it had searched had 100 spaces and not the default space that was 10.
This had to be done everytime there was a Search function
